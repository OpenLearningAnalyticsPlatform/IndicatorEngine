# IndicatorEngine
To run the code :
1.	First Install Intellij IDE.
2.	Open it and select checkout from GITHUB. 
3.	Download apache server.
4.	Configure the location of apache server from within the IDE.
5.	Install a DB server like mysql/mssql.
6.	Change the DB settings in applicationcontext.xml file of the project.
7.	Run the Project.
